# Data-Cleaning-and-Reporting-Automation
A Python-based automation project that cleans raw data by handling missing values, duplicates, and inconsistent data, then generates automated reports and visual summaries using Pandas and Matplotlib.

# Data Cleaning & Reporting Automation

## Project Overview

This project automates the process of cleaning raw data and generating reports using Python and Google Colab.

## Features

- Imported Titanic dataset
- Identified missing values
- Removed duplicate records
- Filled missing values
- Cleaned text data
- Generated summary statistics
- Created visualization
- Exported cleaned dataset

## Technologies Used

- Python
- Google Colab
- Pandas
- Matplotlib

## Output Files

- cleaned_data.csv
- summary_report.csv
- chart.png

## Conclusion

The project demonstrates automated data cleaning, preprocessing, reporting, and visualization techniques that improve data quality and reporting efficiency.
